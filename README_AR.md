# Nabny HR V18.9 Smart Attendance & Payroll

تم إضافة نظام إعدادات الدوام والحضور والبيرول الذكي:

- إعدادات وقت بداية ونهاية الدوام.
- تحديد فترة السماح للحضور والانصراف.
- اختيار أيام العمل من الإعدادات.
- إضافة تاريخ مباشرة العمل لكل موظف.
- احتساب البيرول من تاريخ المباشرة فقط.
- احتساب أيام العمل الفعلية حسب أيام العمل المختارة.
- منع احتساب الغياب قبل تاريخ مباشرة الموظف.
- احتساب التأخير بالدقائق بعد السماحية.
- شاشة تفاصيل حساب داخل البيرول توضح أيام العمل والحضور والغياب وراتب اليوم.
- دعم طريقة قسمة الراتب: أيام العمل الفعلية / 30 يوم ثابت / أيام الشهر.
